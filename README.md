```
### 项目说明
```
1. Spring MVC架构，前后端分离.
2. REST风格接口,响应统一格式的数据.
3. 注解式校验.
4. 全局异常处理.
5. 灵活支持JSON和JSONP.
6. 安全的Redis Token校验，保证接口的安全性.
7. 注解+AOP日志记录
8. 统一数据格式化,如：日期、人民币等.
9. Mybatis-plus做为ＯＲＭ框架.
10. 集成Swagger2接口管理.
